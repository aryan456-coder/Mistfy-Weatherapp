# 🌦️ Mistfy Weather App
Clean and responsive weather app that fetches real-time weather data by city, featuring a modern UI and dynamic updates using the OpenWeather API.

## 🌐 Live Demo

👉 https://mistify-weatherapp.netlify.app/

---

## 🚀 Features

- 🔍 Search weather by city
- 🌡️ Real-time temperature display
- 💨 Humidity and wind speed info
- 🌤️ Dynamic weather conditions
- 🎨 Modern and clean UI
- 📱 Responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- OpenWeather API
